ARCHITECTURE OVERVIEW

ISA-95 自动化金字塔

国际标准 ISA-95 将制造企业系统分为 5 个层级，从现场设备到企业决策逐级向上。

LEVEL 4 · 企业层
ERP · PLM · BI · SRM · CRM
企业资源计划 · 产品生命周期 · 商业智能
LEVEL 3 · 制造运营层
MES / MOM · WMS · QMS · APS · EAM
制造执行 · 仓储管理 · 质量管理 · 排程 · 设备
LEVEL 2 · 监控层
SCADA · HMI · DCS
数据采集与监视控制 · 人机界面 · 集散控制
LEVEL 1 · 控制层
PLC · CNC · DNC · 工业机器人
可编程逻辑控制器 · 数控系统
LEVEL 0 · 现场层
传感器 · 执行器 · 电机 · 传送带 · 扫码枪
物理设备与信号采集
▲ 层级越高数据量越小、决策影响越大 ｜ ▼ 层级越低实时性要求越高

LEVEL 4 · 企业层 · 01

ERP

Enterprise Resource Planning
企业资源计划系统

管理企业核心业务流程的统一平台，涵盖财务、采购、销售、生产、库存、HR 等模块，是数字化建设的"地基"。

代表产品

SAP S/4HANA · Oracle EBS · 金蝶云星空 · 用友 U9 cloud · Microsoft Dynamics 365

核心能力
  • 财务核算 — 总账 / 应收应付 / 资产 / 成本
  • 供应链管理 — 采购到付款 / 订单到收款
  • 生产管理 — 计划 / 排程 / 物料需求 (MRP)
  • 多组织协同 — 集团管控 / 多账簿 / 合并报表
  • 合规审计 — 全流程追溯 / SOX 合规
上游←ERP→下游

SRM / CRM → ERP → MES / WMS

LEVEL 4 · 企业层 · 02

PLM

Product Lifecycle Management
产品生命周期管理

管理产品从概念→设计→工艺→制造→服务→报废的全生命周期数据，是产品数据的"单一真相源"。

代表产品

PTC Windchill · Siemens Teamcenter · Dassault ENOVIA · Aras Innovator

核心能力
  • 文档管理 — CAD 图纸 / 规格书 / 变更记录集中管控
  • BOM 管理 — EBOM → MBOM → SBOM 多视图转换
  • 变更管理 — ECO/ECN 流程：申请→评审→执行→通知
  • 协同设计 — 多部门 / 多地 / 多供应商并行设计
  • 合规管理 — RoHS / REACH / 行业认证
PLM→ERP→MES

物料主数据 / BOM / 工艺路线 → ERP → 下发到 MES 执行

LEVEL 3 · 制造运营层 · 03

MES / MOM

Manufacturing Execution System
制造执行系统

连接 ERP 计划层与车间设备层的桥梁，实时管控生产过程，是智能制造的核心枢纽。

代表产品

Siemens Opcenter · Rockwell FactoryTalk · GE Proficy · SAP ME · 盘古信息 · 赛意

核心能力
  • 工单管理 — 接收生产订单 → 排程 → 派工 → 报工
  • 物料追踪 — 批次 / 序列号全程追溯（正向+反向）
  • 质量管理 — 在线检测 / SPC 统计过程控制 / 不良品处理
  • 设备管理 — OEE 计算 / 设备状态监控 / 维保计划
  • 电子看板 — 实时产量 / 质量 / 设备状态可视化

LEVEL 3 · 制造运营层 · 04

WMS

Warehouse Management System
仓储管理系统

精细化管控仓库作业，优化库存布局与库内流程，是供应链效率的关键环节。

代表产品

Blue Yonder · Manhattan · SAP EWM · 唯智 · 富勒 · Infor WMS

核心能力
  • 入库管理 — 收货 → 质检 → 上架 → 库位分配策略
  • 出库管理 — 波次 → 拣货 → 复核 → 包装 → 发货
  • 库内管理 — 盘点 / 移库 / 补货 / 库龄分析
  • 智能策略 — ABC 分类存储 · 先进先出/后进先出
  • 自动化集成 — AGV / 堆垛机 / 分拣线 / RFID

LEVEL 2 · 监控层 · 05

SCADA

Supervisory Control and Data Acquisition
数据采集与监视控制

连接现场 PLC/传感器，实现设备数据实时采集、可视化监控与远程控制。

代表产品

Siemens WinCC · GE iFIX · Ignition · 力控 · 组态王 · Kepware

核心能力
  • 数据采集 — 设备运行参数 / 产量计数 / 能耗 / 温度压力
  • 实时监控 — 组态画面 / 趋势图 / 报警管理
  • 远程控制 — 设备启停 / 参数下发 / 配方管理
  • 历史数据 — 时序数据库存储 / 历史趋势回溯
  • 协议支持 — Modbus / OPC UA / MQTT / Profinet

LEVEL 2–3 · 跨层 · 06

IIoT 平台

Industrial Internet of Things
工业物联网平台

工业物联网连接设备、系统和人，实现数据汇聚、边缘计算和设备云端管理。

代表产品

PTC ThingWorx · Siemens MindSphere · 树根互联 · 海尔 COSMOPlat · AWS IoT

核心能力
  • 设备连接 — 异构协议适配 / 边缘网关 / 设备模型化
  • 数据湖 — 海量时序数据存储 / 冷热分层 / 数据清洗
  • 规则引擎 — 异常检测 → 自动触发工单 / 报警 / 停线
  • 数字孪生 — 设备/产线/工厂三维可视化与仿真
  • 预测维护 — 设备健康评估 / 剩余寿命预测

LEVEL 3 · 制造运营层 · 07

APS

Advanced Planning & Scheduling
高级计划与排程

在有限产能约束下，通过优化算法自动生成最优生产排程计划，解决"先做哪个、用哪台机、排谁来做"的问题。

代表产品

Asprova · Siemens Preactor · SAP APO · 安达发 · 永凯

核心能力
  • 产能约束排程 — 考虑设备/模具/人员/物料四重约束
  • 多目标优化 — 交期优先 vs 效率优先 vs 成本优先
  • What-if 分析 — 插单/退单/设备故障的快速重排
  • 可视化甘特图 — 每台设备每张工单的起止时间
  • 与 ERP/MES 集成 — ERP 给需求 → APS 排程 → MES 执行

LEVEL 3 · 制造运营层 · 08

QMS

Quality Management System
质量管理系统

覆盖来料、过程、出货、售后四阶段，实现质量策划→控制→改进的 PDCA 闭环。

代表产品

Siemens QMS · IQS · MasterControl · 库得克 · 盈飞无限

核心能力
  • 来料检验 (IQC) — 抽样方案 / 合格判定 / 供应商评分
  • 过程控制 (IPQC) — SPC 控制图 / CPK 分析 / 异常预警
  • 出货检验 (OQC) — 成品抽检 / 合格证管理
  • 不合格品处理 — 8D 报告 / 纠正预防措施 (CAPA)
  • 体系合规 — ISO 9001 / IATF 16949 / GMP

LEVEL 3 · 制造运营层 · 09

EAM

Enterprise Asset Management
设备资产管理系统

对全厂设备资产进行全生命周期管理，从采购、安装、运维到报废，提升设备综合效率 OEE。

代表产品

IBM Maximo · SAP PM · Infor EAM · 用友 AIoT · 鲁邦通

核心能力
  • 设备台账 — 设备编码 / 规格 / 位置 / BOM 结构
  • 预防性维护 — 点检计划 / 润滑计划 / 周期性保养
  • 故障维修 — 报修 → 派工 → 维修 → 验收闭环
  • 备件管理 — 安全库存 / 采购申请 / 以旧换新
  • OEE 分析 — 时间开动率 × 性能开动率 × 合格率

LEVEL 4 · 企业层 · 10

SRM & CRM

Supplier / Customer Relationship Management
供应商 & 客户关系管理

管理上下游伙伴关系，打通从供应商到客户的端到端价值链。

SRM 供应商管理
  • 寻源与准入 — 供应商注册 / 评估 / 准入 / 分级
  • 采购协同 — 订单协同 / 送货协同 / 对账协同
  • 绩效管理 — QCD 评分 / 8D 整改 / 淘汰机制
CRM 客户管理
  • 销售管理 — 线索 → 商机 → 报价 → 合同
  • 服务管理 — 售后工单 / SLA / 客户满意度
  • 营销自动化 — 客户画像 / 精准营销

LEVEL 4 · 决策层 · 11

BI & 数字孪生

Business Intelligence & Digital Twin
商业智能 & 数字孪生

将全链路数据转化为决策洞察，用虚拟模型映射物理世界，是智能工厂的"大脑"。

BI 商业智能
  • 多系统数据 ETL 汇聚 → 数据仓库 / 数据湖
  • 管理驾驶舱：OEE / 产量 / 良率 / 能耗一屏展示
  • 自助分析：拖拽式报表 / 多维下钻 / 预警推送
  • Power BI · FineBI · Tableau · 帆软
数字孪生
  • 3D 虚拟工厂 → 1:1 还原产线 / 设备 / 物流
  • 实时数据驱动 → 虚拟与物理同步运行
  • 仿真推演 → 新产线规划 / 瓶颈分析 / What-if
  • Unity · Unreal · NVIDIA Omniverse · 优锘

关键成功要素

① 主数据先行：物料编码 / BOM / 工艺路线标准化是所有系统的基础 ··· ② 流程先于系统：先梳理优化业务流程，再选型匹配的系统 ··· ③ 集成规划：提前设计 ESB/API 架构，避免后期形成新的数据孤岛 ··· ④ 一把手工程

SUMMARY

总结

制造业数字化转型不是单一系统能完成的，而是
ERP 管资源 ＋ PLM 管产品 ＋ MES 管执行 ＋ WMS 管仓储 ＋
SCADA 采数据 ＋ APS 优排程 ＋ QMS 保质量 ＋
EAM 管设备 ＋ IoT 联万物 ＋ BI 助决策

共同构成一个完整的数字化制造生态系统。
